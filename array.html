Arrays/ArrayLists

What are Arrays?

Arrays are used to store many instances of a specific data type. They can easily represent a large list of objects. After initialization, the size of the array cannot be changed. The following code creates an integer array named arr, with size 50.

int arr [] = new int [50];

Arrays have indicies, which point to different locations in an array. To modify an index:

arr[0] = 4;

In this example, index 0 now has the value 4.

To access an index:

System.out.println(arr[0]);

In this example, since index 0 has the value 4, 4 is outputted.

Indices start at 0 and end at the size of the array - 1. You can make an array of strings, integers, doubles, and any custom class you choose.

Note: if you access an index out of the bounds of the array, you get an IndexOutOfBoundsException.

What are ArrayLists?

ArrayList is a class found in java.util, which allows for the creation of an array with mutable size. This means objects can be added and removed from the ArrayList, which changes the size. To use ArrayLists, you must first import java.util.ArrayList.

import java.util.ArrayList;

Then, you must create an ArrayList object:

ArrayList<Integer> arr = new ArrayList<Integer>();

You may also choose to set the size of the ArrayList in the initialization:

ArrayList<Integer> arr = new ArrayList<Integer>(50);

In this example, the size of the ArrayList is 50.

ArrayList Methods

There are many methods that can be used to interact with ArrayLists. Below, a few of them will be covered.

add

The add method allows you to insert elements inside the ArrayList. The method is overloaded, so there are two usages shown below.

arr.add(50);

The number 50 is added to the back of the ArrayList.

arr.add(2, 50);

The number 50 is added to index 2 in the ArrayList, pushing the previous elements back.

size

Calling 

arr.size();
will return the size of the ArrayList.

remove

Calling 

arr.remove(2);
will remove the element at index 2 in the ArrayList. All elements from index 3 and beyond are shifted.

get

This method will return the element found at a specified index.

arr.get(2);

This example would return the element at index 2.

Note: if you access an index out of the bounds of the ArrayList, you get an IndexOutOfBoundsException.
