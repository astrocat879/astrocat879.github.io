Final Project

Our final project was made in JavaFX, a popular GUI library for Java. In this tutorial, I won't be covering all the details about FX. Instead, I will talk about how to get started creating a game.

Before creating your project, you need to design what the game will look like. In our case, we drew out all the screens beforehand.

After planning all the screens, you should find which GUI suits your game the most. This can be determined by looking at your skillset, and searching online. In our case, we were interested in learning JavaFX.

During the programming phase, you will inevitably face troubles, but most should be easily solved with Google. If you are working with others, you should set up a Github repository, so that all changes can be stored, and everyone can collaborate without hassle.
