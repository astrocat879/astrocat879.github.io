OOP

What is OOP?

OOP stands for object oriented programming. In Java, this means creating classes.

OOP is used for creating your own data structures. It allows programmers to define a set of properties and behaviours to represent objects. Below is an example of a class: 

public class Cat{
}

Properties

Properties, such as age or size, are present in all classes in the form of instance variables. Instance variables are variables which are present in all objects of a class, but can have different values. The the following example, we will make 2 integers, representing a cat's age and size. 

public class Cat{
    private int age, size;
}

Constructor

In order to create objects/instances of a class, we need a class constructor. 

public class Cat{
    private int age, size;
    public Cat(int age, int size){
        this.age = age;
        this.size = size;
    }
}

With this constructor, we can now create objects, which are instances of the class.

Cat bob = new Cat(5, 10);
Cat abby = new Cat(6, 20);

Behaviours

Behaviours are common actions that can be performed by all objects of a class. We can modify the Cat class to show this.

public class Cat{
    private int age, size;
    public Cat(int age, int size){
        this.age = age;
        this.size = size;
    }
    public int getAge(){
        return age;
    }
    public void setAge(int age){
        this.age = age;
    }
}

Now, the age of the cat can be modified, and you can retrieve the age through a method call.

Conclusion

To create a primitive class, all you need are properties, behaviours, and a constructor. Have fun with OOP!
